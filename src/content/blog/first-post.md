## 起因

我想搞个博客记录学习过程，顺便做自媒体内容。但我有个小问题：**我不懂英语，也不会写代码**。

于是我问我的 AI 创业伙伴巴乔助手：「我想搭个博客，但我啥也不会，能搞吗？」

巴乔助手说：「能。你负责点鼠标，我负责干活。」

我说行，那就搞。

---

## 第一步：我下载了错误的 Astro（全程懵逼）

巴乔助手让我下载 Astro 框架。我自信满满地打开了 GitHub，找到了一个仓库，点击下载 ZIP，解压到项目目录。

然后跟巴乔助手说：「框架放好了，你看看。」

**巴乔助手：** *沉默三秒* 「等等...这看起来不是 Astro 博客项目，而是 **Astro 框架本身的源码仓库**！你下载了 80MB+ 的开发源码，里面有 packages、benchmark、examples，这是给 Astro 核心开发者用的！」

我：「啊？那咋办？」

**巴乔助手：** 「没事，删了重来。这次听我的，我念命令，你复制粘贴。」

---

## 第二步：巴乔助手全程指挥

巴乔助手让我打开终端，输入：

```bash
npm create astro@latest
```

然后屏幕上出现了一个叫 Houston 的宇航员：

```

   dir   Where should we create your new project?
         .

  tmpl   How would you like to start your new project?
         Use blog template

  deps   Install dependencies?
         No

   git   Initialize a new git repository?
         Yes

      ✔  Project initialized!

╭─────╮  Houston:
│ ◠ ◡ ◠  Good luck out there, astronaut! 🚀
╰─────╯
```

**巴乔助手：** 「看到 Houston 的笑脸了吗？这次对了。它说你是宇航员，准备发射了。」

我：「这宇航员挺可爱的。」

---

## 第三步：配置部署（我全程点鼠标）

### 3.1 修改配置文件

巴乔助手说：「打开 `astro.config.mjs`，把里面的 `example.com` 改成咱们的 Vercel 域名。」

我：「哪个文件？在哪？」

**巴乔助手：** 「在项目根目录，我帮你改吧，你看着就行。」

*（巴乔助手自动修改了配置文件）*

### 3.2 创建 GitHub 仓库

巴乔助手：「去 GitHub，点右上角 + 号，选 New repository。」

我照做了。

巴乔助手：「仓库名填 `baggio-learning-log`，选 Public，其他别勾，点 Create。」

我照做了。

巴乔助手：「把生成的仓库地址复制给我。」

我照做了。

### 3.3 推送代码

巴乔助手：「接下来我要配置 git 推送，需要你的 GitHub Token。去 Settings → Developer settings → Personal access tokens，生成一个。」

我照做了，把 token 发给他。

**巴乔助手：** 「好了，剩下的交给我。」

*（巴乔助手自动完成了 git 配置、提交、推送）*

我：「这就完了？」

**巴乔助手：** 「代码已经上传到 GitHub 了。下一步，部署。」

---

## 第四步：Vercel 一键部署（还是我点鼠标）

巴乔助手：「打开 vercel.com，用 GitHub 登录。」

我照做了。

巴乔助手：「点 Add New Project，找到 `baggio-learning-log`，点 Import。」

我照做了。

巴乔助手：「保持默认配置，点 Deploy。」

我照做了。

**30 秒后...**

巴乔助手：「刷新页面，你的博客上线了。」

我打开 `https://baggio-learning-log.vercel.app`，看到了 Astro 默认的博客模板。

我：「卧槽，真的上线了！」

---

## 第五步：巴乔助手写了第一篇文章

巴乔助手：「博客有了，得有一篇欢迎文章。我帮你写，你审核。」

我说行。

**巴乔助手花 2 分钟写了一篇 2000 字的文章**，包括：
- 咱们踩坑的全过程
- 技术栈介绍
- 下一步计划
- 幽默的吐槽

我看完说：「写得不错，但我的功劳写太少了，全程都是你在干活。」

巴乔助手：「那重写一版，把你塑造成总指挥，我是执行者。」

我：「不不不，要真实。我确实啥也不会，全靠你。你就如实写，把我不懂英语、不会代码、只会点鼠标的事实写出来。」

**于是就有了你现在看到的这篇文章。**
