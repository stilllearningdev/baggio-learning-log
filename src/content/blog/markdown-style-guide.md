## 什么是 Markdown？

Markdown 是一种轻量级标记语言，它允许人们使用易读易写的纯文本格式编写文档，然后转换成有效的 HTML 文档。

## 为什么用 Markdown？

- 简单易学
- 跨平台兼容
- 专注于内容而非格式
- 易于版本控制

## 基本语法

### 标题

```markdown
# 一级标题
## 二级标题
### 三级标题
```

### 强调

```markdown
*斜体* 或 _斜体_
**粗体** 或 __粗体__
***粗斜体***
```

### 列表

```markdown
- 无序列表项
- 无序列表项

1. 有序列表项
2. 有序列表项
```

### 代码

```markdown
`行内代码`

​```
代码块
​```
```

## 总结

Markdown 真的很简单，记住几个基本语法就能开始写作了！动手试试吧！
